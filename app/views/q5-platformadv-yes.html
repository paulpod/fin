Short "How to...?" guide on using advanced platform features.

Chart Profiles

Chart profiles store the exact apperance, time frame and indicators used on a chart so you can quickly apply them to another chart.


Generic image of a customised chart

  1. Modify the chart as desired, including any indicators you want saved in the profile.
  2. Select the Save current profile option from the drop down menu and enter the name you want to call it.
  3. All saved profiles now appear in the drop down menu.
