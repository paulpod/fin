Tabular numbers should be aligned right, and where possible displayed using a monospace variant of the font. Many modern browsers can force this using CSS:
-webkit-font-feature-settings: 'tnum';

Table headings should be smaller and remain aligned left. Stripes on tables help with readabity, especially long, wide rows. Stripes remove the need for lines, which reduces overall visual noise.
