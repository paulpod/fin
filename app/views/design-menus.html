Menus

Menus contain options and links to expand the choices available from a specific point on a page or form.

Menus should not be a dumping ground for random functions, but should be a coherent selection of similar or related options.

Generally avoid using hidden menus, rethink the needs and approach the problem using visible options first.

Menus need to offer affordance, that is if it reacts to being swiped open, there should be an appropriate handle that invites this.

If a menu appears from a button or link, it should offer appropriate 'button-ness' to invite this.

Menus should not be hidden without explanation. Menus implemented as drop-down lists (aka select boxes) are generally poor choices for accessibility for lots of reasons.

Using select/option drop downs are sometimes painful for users and the behaviour may not match the actions desired, especially on mobile platforms.
